# Graduation-Form
## Uses Node.JS, Express.JS, and MYSQL  
.env file is needed for the variables:
  *MYSQL_HOST  
  *MYSQL_USER  
  *MYSQL_PASSWORD  
  *MYSQL_DATABASE  
  *MYSQL_PORT  
  *RSESSION_SECRET  
  *LSESSION_SECRET  

Run npm install to download packages and run npm start to start on localhost 3000
